Para compilar la clase Java, abre una terminal y navega hasta el directorio que contiene el archivo .java. Luego, ejecuta el siguiente comando:

javac FicheroTXT.java

Esto compilará la clase y creará un archivo .class en el mismo directorio.

Para ejecutar el archivo .class desde la línea de comandos, utiliza el siguiente comando:

java FicheroTXT (Aqui va la ruta C:\Users\santi\eclipse-workspace\ItAcademy\IT_Academy) > lista.txt.
